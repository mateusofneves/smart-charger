# Descrição dos Diretórios

## 📂 assets

Contém imagens e evidências da execução do protótipo.

operacao-2.png → Cenário normal sem uso dos potenciometros para simulação.
operacao-3.png → Cenário com utilização do potenciometro solar.
operacao-4.jpeg → Cenário com utilização do potenciometro teste.
